# How AlphaYield Works

AlphaYield operates through multiple protocol layers that optimize user rewards and ecosystem sustainability.

---

## 1. Farming Layer

Users provide liquidity to supported pools and receive:

* Farming rewards
* LP incentives
* Ecosystem emissions

The farming system is optimized for:

* Long-term liquidity
* Dynamic reward distribution
* Sustainable APY generation

---

## 2. Staking Layer

Users can stake AYT tokens to:

* Earn protocol rewards
* Access governance
* Receive boosted farming incentives

The staking model supports:

* Flexible staking
* Locked staking
* Reward multipliers

---

## 3. Auto-Compounding Vaults

AlphaYield vaults automatically:

* Harvest rewards
* Reinvest earnings
* Optimize compounding strategies

This helps users maximize yield efficiency without manual management.

---

## 4. Governance Layer

AYT holders participate in:

* Protocol governance
* Treasury proposals
* Reward adjustments
* Ecosystem decisions

This creates a decentralized community-driven ecosystem.

---

## 5. Analytics Layer

The protocol provides:

* TVL monitoring
* APY tracking
* Reward analytics
* Vault performance metrics
